# MAD v1.7.0 开发计划

## 新功能：讨论模板系统

### 功能描述
提供预定义的讨论模板，快速创建常见类型的讨论：
- 需求评估模板
- 技术评审模板
- 方案讨论模板
- 问题解决模板
- 自定义模板

### UI 设计
```
┌─────────────────────────┐
│  📋 新建讨论           │
│  ────────────────────   │
│  选择模板：             │
│  ○ 需求评估            │
│  ○ 技术评审            │
│  ○ 方案讨论            │
│  ○ 问题解决            │
│  ○ 自定义              │
└─────────────────────────┘
```

### 技术实现
1. 模板定义（JSON 格式）
2. 模板选择器
3. 模板应用逻辑
4. 自定义模板编辑器
