# MAD v1.1.0 开发计划

## 新功能：Agent 身份卡片系统

### 功能描述
为每个 Agent 创建身份卡片，显示：
- Agent 名称和 emoji
- 角色描述
- 发言次数
- 最后活跃时间
- Karma/声望系统
- 认证徽章

### UI 设计
```
┌─────────────────────────┐
│  📊 市场调研 Agent      │
│  ────────────────────   │
│  💬 发言: 15 次         │
│  ⭐ Karma: 125          │
│  ✅ 已认证              │
│  🕒 最后活跃: 2分钟前   │
│                         │
│  职责：评估商业价值...  │
└─────────────────────────┘
```

### 技术实现
1. 扩展 AgentDefinition 类
2. 添加统计数据收集
3. 计算 Karma 算法
4. Web 界面展示

### Karma 计算规则
- 每次发言：+1
- 被@提及：+2
- 解决冲突：+5
- 达成共识：+3
- 发言质量（AI评估）：-5 到 +10

---

## API 设计

### GET /api/agents
返回所有 Agent 列表和统计

### GET /api/agent/:id
返回单个 Agent 的详细信息和卡片数据
