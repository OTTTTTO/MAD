# MAD v1.5.0 开发计划

## 新功能：多讨论组管理和快速切换

### 功能描述
支持同时管理多个讨论组，快速切换：
- 讨论组标签页
- 快速键切换
- 讨论组固定
- 讨论组归档

### UI 设计
```
┌─────────────────────────────────┐
│ [讨论1] [讨论2] [讨论3] [...]  │
├─────────────────────────────────┤
│                                 │
│  当前讨论内容                   │
│                                 │
└─────────────────────────────────┘
```

### 技术实现
1. 前端状态管理
2. 讨论组缓存
3. 快速键绑定
4. 本地存储
