# MAD v1.4.0 开发计划

## 新功能：主题定制和响应式优化

### 功能描述
让用户可以自定义界面主题，优化移动端体验：
- 浅色/深色主题切换
- 自定义颜色方案
- 响应式布局优化
- 移动端适配

