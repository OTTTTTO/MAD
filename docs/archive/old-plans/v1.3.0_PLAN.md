# MAD v1.3.0 开发计划

## 新功能：搜索和过滤

### 功能描述
用户可以搜索和过滤讨论内容：
- 按关键词搜索讨论
- 按角色过滤消息
- 按时间范围筛选
- 按状态筛选（进行中/已结束）

### UI 设计
在侧边栏添加搜索框和过滤器：
```
┌─────────────────────────┐
│  🔍 搜索讨论...         │
│  ────────────────────   │
│  过滤器：               │
│  ☑ 进行中              │
│  ☑ 已结束              │
│  ☐ 只有我参与的        │
└─────────────────────────┘
```

### 技术实现
1. 前端搜索和过滤
2. 后端搜索 API（全文搜索）
3. 消息内容索引
4. 高亮搜索结果

---

## API 设计

### GET /api/search?q=keyword
搜索讨论和消息

### GET /api/discussions?status=active&role=market
过滤讨论列表
