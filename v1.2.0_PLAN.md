# MAD v1.2.0 开发计划

## 新功能：导出讨论记录

### 功能描述
用户可以导出讨论组的完整记录，支持多种格式：
- Markdown (.md)
- JSON (.json)
- PDF（后期）

### UI 设计
在讨论详情页面添加"导出"按钮：
```
┌─────────────────────────┐
│  [导出 ▼]              │
│  ├─ Markdown           │
│  ├─ JSON               │
│  └─ PDF (即将推出)      │
└─────────────────────────┘
```

### 技术实现
1. 后端 API 生成导出文件
2. 前端下载按钮
3. Markdown 格式化
4. JSON 结构化

---

## API 设计

### GET /api/discussion/:id/export/:format
参数：
- id: 讨论 ID
- format: markdown | json

返回：文件下载
