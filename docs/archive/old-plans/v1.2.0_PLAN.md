# MAD v1.2.0 开发计划（更新）

## 新功能：实时 WebSocket 推送

### 当前问题
- 使用轮询（每 5 秒）更新数据
- 不够实时
- 浪费资源

### 解决方案
使用 WebSocket 实现实时推送：
- 新消息立即推送
- Agent 统计实时更新
- 讨论状态变更通知

### 技术实现
1. 添加 ws 库
2. WebSocket 服务器
3. 前端 WebSocket 客户端
4. 实时消息推送
