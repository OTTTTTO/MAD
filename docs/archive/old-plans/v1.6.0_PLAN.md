# MAD v1.6.0 开发计划

## 新功能：讨论统计和分析

### 功能描述
为每个讨论组提供详细的统计数据：
- 消息数量趋势
- Agent 参与度
- 讨论活跃度曲线
- 关键词云
- 时间分布

### UI 设计
```
┌─────────────────────────┐
│  📊 讨论统计           │
│  ────────────────────   │
│  消息数: 156            │
│  参与者: 6              │
│  时长: 45分钟           │
│  最活跃: 📊 市场调研    │
└─────────────────────────┘
```

### 技术实现
1. 统计数据计算
2. 图表渲染（使用轻量级库）
3. 数据聚合
4. 时间序列分析
